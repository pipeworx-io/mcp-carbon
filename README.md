# mcp-carbon

Carbon MCP — UK Carbon Intensity API (free, no auth)

Part of the [Pipeworx](https://pipeworx.io) open MCP gateway.

| Tool | Description |
|------|-------------|

## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "carbon": {
      "url": "https://gateway.pipeworx.io/carbon/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use carbon
```
